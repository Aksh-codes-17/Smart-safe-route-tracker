 Smart Safe Route Planner 🚦
 📌 Description
An AI-based system that finds the safest and most efficient route between locations using:
- Distance
- Crime rate
- Lighting conditions
 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Flask (Python)
- Algorithm: Graph + Shortest Path (NetworkX)
▶️ How to Run
Backend
cd backend
python app.py
 Frontend
Open index.html in browser

 📊 Future Improvements
- Real-time data (Google Maps API)
- ML-based safety prediction
- Live traffic integration
